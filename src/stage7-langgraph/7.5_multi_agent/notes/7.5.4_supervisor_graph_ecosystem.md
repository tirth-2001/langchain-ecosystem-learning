# ✅ **7.5.4 — Supervisor Graph (Manager Agent Dispatching Multiple Specialists)**

### **Use Case: Customer Support Ticket Resolver**

We’ll follow the 5-part structure.

---

# **1) GOAL**

Build a **Supervisor Graph** where:

```
User Ticket → Classifier Agent → Supervisor →
     ├── Troubleshooter Agent
     ├── Billing Agent
     ├── Escalation (Human)
     └── Finisher Agent
```

🎯 **Objectives:**

- Supervisor decides _which specialist agent_ handles the ticket
- Supports multiple turns
- Allows escalation to human
- Produces a final, professional customer-support reply
- Demonstrates advanced graph routing

---

# **2) THEORY — Concepts Behind Supervisor Graph**

### **a) Supervisor Pattern**

A **Supervisor Agent** evaluates the current state and decides the next node:

```
supervisorNode(state):
    if state.category == 'technical':
        return 'troubleshooter'
    if state.category == 'billing':
        return 'billingAgent'
    if state.needs_escalation:
        return 'escalation'
    return 'finisher'
```

It behaves like:

- CrewAI’s "Manager Agent"
- AutoGPT’s decision planner
- Human support routing logic

---

### **b) Specialist Agents**

Each agent is a separate node:

| Agent              | Role                                       |
| ------------------ | ------------------------------------------ |
| **Classifier**     | Categorizes: technical / billing / general |
| **Troubleshooter** | Fixes device/app/network issues            |
| **Billing Agent**  | Refunds, payment issues, invoices          |
| **Escalation**     | HITL node → lets human decide              |
| **Finisher**       | Writes final polished response             |

---

### **c) Shared State Structure**

State contains:

```ts
{
  ticket: string,
  category: string,
  analysis: string,
  resolution: string,
  needs_escalation: boolean,
  final: string
}
```

Supervisor reads + updates these fields.

---

### **d) Graph Routing**

We will use:

- `addNode`
- `addEdge`
- `addConditionalEdges`

Supervisor decides which path to take → dynamic routing.

## **3.3 — Classifier Agent**

```ts
const classifierPrompt = ChatPromptTemplate.fromTemplate(`
Classify this customer ticket into one category:
- "technical"
- "billing"
- "general"
Return ONLY the category.

Ticket: {ticket}
`)

const classifyChain = RunnableSequence.from([classifierPrompt, llm, (out) => out.content.trim().toLowerCase()])

async function classifierNode(state) {
  const category = await classifyChain.invoke({ ticket: state.ticket })
  console.log('🟦 Category:', category)
  return { category }
}
```

---

## **3.4 — Troubleshooter Agent**

```ts
const troubleshootPrompt = ChatPromptTemplate.fromTemplate(`
User ticket: {ticket}

Provide a step-by-step solution for TECHNICAL issues.
If outside technical scope → return "ESCALATE".

`)

const troubleshootChain = RunnableSequence.from([troubleshootPrompt, llm, (out) => out.content.trim()])

async function troubleshooterNode(state) {
  const resolution = await troubleshootChain.invoke(state)
  const needs_escalation = resolution.includes('ESCALATE')
  return { resolution, needs_escalation }
}
```

## **3.5 — Billing Agent**

```ts
const billingPrompt = ChatPromptTemplate.fromTemplate(`
User ticket: {ticket}

Handle billing, payment, refund, invoice issues.
If issue cannot be resolved → return "ESCALATE".
`)

const billingChain = RunnableSequence.from([billingPrompt, llm, (out) => out.content.trim()])

async function billingNode(state) {
  const resolution = await billingChain.invoke(state)
```

---

## **3.8 — Supervisor Node**

```ts
async function supervisorNode(state) {
  console.log('🟪 Supervisor routing...', state.category)

  if (state.needs_escalation) return 'escalation'

  switch (state.category) {
    case 'technical':
      return 'troubleshooter'
    case 'billing':
      return 'billing'
    default:
      return 'finisher'
  }
}
```

---

## **3.9 — Graph Wiring**

```ts
const graph = new StateGraph(StateAnnotation)
  .addNode('classifier', classifierNode)
  .addNode('troubleshooter', troubleshooterNode)
  .addNode('billing', billingNode)
  .addNode('escalation', escalationNode)
  .addNode('finisher', finisherNode)
  .addNode('supervisor', supervisorNode)

  .addEdge(START, 'classifier')

  .addEdge('classifier', 'supervisor')

  .addConditionalEdges('supervisor', (state) => state.category, {
    technical: 'troubleshooter',
    billing: 'billing',
    general: 'finisher',
  })

  .addConditionalEdges('troubleshooter', (state) => (state.needs_escalation ? 'escalate' : 'done'), {
    escalate: 'escalation',
    done: 'finisher',
  })

  .addConditionalEdges('billing', (state) => (state.needs_escalation ? 'escalate' : 'done'), {
    escalate: 'escalation',
    done: 'finisher',
  })

  .addEdge('escalation', 'finisher')

  .addEdge('finisher', END)
```
